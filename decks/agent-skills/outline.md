# Agent Skills: Extiende tu Agente de IA

## Slide 1: Title Slide

- **Title:** Agent Skills
- **Subtitle:** Cómo extender las capacidades de tu agente de IA
- **Author:** Jose Luis

## Slide 2: El Problema

- Los agentes de IA tienen capacidades limitadas
- No conocen procesos específicos del equipo
- Cada proyecto requiere conocimientos únicos

## Slide 3: Nuestra Solución

- Skills modulares y reutilizables
- Instalación con un solo comando
- Comunidad activa de developers

## Slide 4: Qué son los Skills

- Plugins que-extienden el agente
- Conocimiento procedimental
- Definidos en archivos SKILL.md

## Slide 5: Características Principales

- Instalación simple: npx skills add
- Multi-Agente: Copilot, Cursor, Claude, etc.
- Formato abierto y estándar

## Slide 6: Casos de Uso

- Commits convencionales
- Creación de presentaciones
- Revisiones de código
- Despliegues automatizados

## Slide 7: Comunidad

- 91K+ installs únicos
- 50+ skills disponibles
- forks activos

## Slide 8: Cómo Funciona

1. Busca el skill que necesitas
2. Instala con npx skills add
3. Usa el skill con tu agente
